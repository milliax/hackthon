# V-SDGs

### The volunteers-matching project for UN SDGs

## Installation
1. Install Dependencies
```shell
npm install
```

2. Run the application in local mode
```shell
npm run dev
```

## Dockerized build
```shell
docker run -idt -p 3000:3000 hsuan1117/v-sdgs.fe:latest
```

## Configuration
必須將.env中的API金鑰改為自己的，否則會Google Map無法正常顯示
